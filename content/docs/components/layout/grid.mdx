## API Reference

This component is based on the `div` element and supports [common margin props](/themes/docs/theme/layout#margin-props).

Use these props to create grid layouts.

The following props are shared between [Box](./box), [Flex](./flex), [Grid](./grid), [Container](./container) and [Section](./section) layout components.
